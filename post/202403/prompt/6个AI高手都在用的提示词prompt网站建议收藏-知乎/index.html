6个AI高手都在用的提示词prompt网站，建议收藏！ - 知乎 -- 知识铺

2024年3月18日
| 阅读

你是不是在网上看各种AI教程时，发现别人的提示词总是很优质，得到的回复质量也更好。明明大家用的同一个工具，其他人就是比你优秀。其实站在巨人的肩膀上学习，会比一个人苦苦摸索效率来得更高更快。

以ChatGPT为例，回复内容的质量取决于提示词（即 Prompt）。如果你想得到高质量的回答，那么你就需要提供高质量的Prompt。接下来，就给大家推荐几个prompt提示词网站，以便AI模型能够准确地理解你的需求和意图。

1、FlowGPT

传送门：http://flowgpt.com

FlowGPT是一个集分享、发现和学习ChatGPT提示词于一体的在线平台。他们还会偶尔举办提示词黑客马拉松活动，让用户有机会通过提交优秀提示词赚取奖励。

FlowGPT也提供了大量的资源和教程，帮助用户了解并掌握如何使用和创作ChatGPT提示词。无论是初学者还是有经验的开发者，都能在这里找到适合自己的学习资料。 2、Bestprompts

传送门：http://bestprompts.cn

Bestprompts是一个全行业AI会话辅助工具，为各行业的内容创作者提供全面准确的提示语，帮助用户高效的和AI对话，以期在最短时间、最短路径达成内容创作目标。

按照职业和主题查看适用于chatGPT的提示词，方便你根据自己的需求找到适合的提示词。 3、Hugging Face

传送门：http://huggingface.co

Hugging Face Hub 是一个探索、实验、合作，并建立机器学习技术的中心场所。在这里任何人都可以分享和探索模型、数据集等，大家轻松合作，共同构建机器学习模型，Hugging Face Hub 也因此被称做「机器学习之家」。

打开ChatGPT Prompt Generator，输入自己想要的gpt角色，点击submit，就可以生成一段专属你的提示词。 4、ai-prompt

传送门：hero.page/ai-prompts

这是一个很全的在线提示词生成器，包括广告副本生成、图像和后期创意生成器、自动化脚本等，同时还可以创建自己的提示库。

如果你想在某一行业深入使用不同方式，产生不同效果的文案，那这个很适合你。它可以教你从故事，情感，幽默等等不同角度书写提示词。 5、Prompthero

传送门：http://prompthero.com

PromptHero支持多种流行的AI模型，包括Stable Diffusion、Midjourney、DALL-E、ChatGPT等，用户可以根据自己的喜好和需求选择不同的模型来生成图像、文本和音乐。

PromptHero还提供了一个丰富的提示库，用户可以浏览、收藏、评价和分享其他用户创建的提示，也可以上传自己的提示和作品，与其他AI艺术爱好者交流互动。 6、AiShort

传送门：www.aishort.top

一个简单易用的ChatGPT快捷指令表，旨在提高用户的生产力。网站提供了一系列的ChatGPT快捷指令，涵盖了各种领域，如写作辅助、文章/报告、IT/编程等。

用户可以通过标签筛选、关键词搜索和一键复制Prompts的方式，快速找到他们需要的ChatGPT快捷指令。这些指令可以帮助用户更好地利用ChatGPT，提高他们的工作效率和生产力。
