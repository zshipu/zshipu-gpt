有抱负或专业的数据科学家，你非常了解大量的SQL命令，算法，Excel函数和谷歌搜索进入你的工作。虽然您每天使用的那些都致力于内存，但有时您可能需要跳入需要新工具或所选编程语言的不同应用程序的项目。

输入：数据科学备忘单的终极汇编。这是您的首选资源，可让您在一个地方查看最常见的公式和算法，而不是在 Google 结果页面中筛选您需要的答案。

这些链接将有助于加快您的编程和计算速度，以便您可以继续解决更大的问题。请继续阅读，找到有关 Python、SQL、机器学习甚至线性代数的最有用的参考指南。当您需要复制/粘贴一行 Python 或查找乘法矩阵的规则时，我们为您提供了帮助。

机器学习

机器学习正在改变我们的世界，数据科学家正在推动这一变化。我们的虚拟助手、Instagram算法和谷歌结果都依赖于机器学习。但是有很多编程可以创建消费者最终每天与之交互的机器学习模型。它始于庞大的数据集和大量巧妙的编程。

人工智能

借助人工智能，数据科学家正在实现我们以前认为不可能实现的壮举。就在几年前，计算机做出决策、从数据中学习和准确预测未来的想法听起来比科学更科幻。时代变了：医疗保健、教育、金融和其他行业的公司现在依靠人工智能来改进他们的产品并更好地了解他们的客户。

人工智能的能力正在快速发展，但基础将保持不变。我们收集了一些人工智能世界中最常用的术语和算法。快速提醒不同神经网络的比较方式，或研究最常见的 AI 模型。

.SQL

世界各地的数据科学家都使用 SQL 来组织表中的数据或使用多个数据集。SQL通常用于提取给定分析所需的数据，而Python及其许多专用库随后被引入来完成繁重的工作（更多内容将在后面）。

Python

Python是最容易上手的编程语言之一，但它也是数据分析最有用的语言之一。它加快了清理和操作大型数据集的过程，使Excel功能尘埃落定。

由于Python对大数据领域至关重要，因此现在有大量专门为数据分析构建的补充工具。像NumPy，SciPy和Pandas这样的库加速了Python的处理能力，使它们对数据科学家来说是无价的，也是我们列表的一大亮点。Sci-kit Learn特定于机器学习，而Bokey和Matplotlib可用于可视化数据。每个的基本功能都在下面的备忘单中。

R

这种编程语言的设计只有一个目标：清理、分析和直观地表示数据。当科学家需要绘制从数千甚至数百万用户中提取的数据时，R 允许他们只需几行强大的代码即可完成，从而节省数小时繁琐的电子表格劳动力。虽然这是更广泛的计算机编程世界中鲜为人知的语言之一，但由于显而易见的原因，它是数据科学家的最爱。

数学

显而易见的是：数据科学是一个非常复杂的领域，需要一些相当高级的数学。根据您的专业领域，您可能需要定期使用微积分、线性代数和统计学。为了在该领域取得真正的进展，数据科学家需要全面了解工作中的概念，以及它们如何在不同场景中应用。

如果你一开始就决定从事数据科学，你可能在数学方面有坚实的基础。这些工具并非旨在从头开始教授微积分或代数，也不是为初学者传授对这些概念的真正理解。它们是数据科学学生和专业人士的资源，可帮助您快速找到特定方程式或检查您的工作。

即使对于经验丰富的数据科学家来说，其中一些公式也会变得有点模糊——特别是如果你不是每天都使用它们的话。这些是一目了然的参考工具，其中包含数据科学家可能遇到的所有方程式和定义。
