ChatGPT刚刚从一个工具变成了一个平台

2023年3月27日
| 阅读

聊天GPT刚刚进化！好吧，不是以“我变得有知觉！”的方式，而是“我已经从一个勉强成为一个工具，变成了一个平台！”的方式。我们每天都听到世界上关于人工智能的新消息。它突然处于软件开发和技术领域的最前沿。从微软在其所有办公产品中引入副驾驶几乎紧随其后，谷歌对其文档做了同样的事情，人工智能领域发生了很多事情，我们可以称之为“三月疯狂”。谷歌开放了BARD以与其竞争对手不断增长的人气竞争。从高盛（Goldman Sachs）到MIRO，从Mozilla到Adobe，这个星球上的每家公司都在争先恐后地投资人工智能。

但开放 AI 继续取得长足进步，这两项进步值得您关注！

GPT-4 上线

14 月 4 日，Open AI 发布了可以说是最先进的大型语言模型 GPT-4 的最新版本。拥有 ChatGPT Plus 订阅的用户可以访问它，其他人可以加入等候名单。必应还在幕后使用 GPT-<>。它进一步推动了深度学习的扩展边界。与任何其他工具一样，新版本有望更加先进，并且在各种情况下的基准测试结果中突出显示了这一点。开放AI也给了它一双眼睛，*变形！*聊天 GPT 现在可以接受视觉输入，人们已经在对其进行测试。给它看一张冰箱的照片，它可以帮助你计划你的饭菜！

GPT-4 在 Azure AI 超级计算机上接受了一年的培训。虽然它的基准测试显示出它优于它的前辈，但它并非没有缺陷。GPT-3 和 GPT-4 能力之间的确切差异可能很难衡量。OpenAI承认，差异是“微妙的”。他们运行了多个测试和场景，GPT-4 似乎全部超过了。它仍然不是没有“幻觉”，尽管它可能会产生40%的事实正确结果。它也更有可能避免受限制的内容。

OpenAI还发布了OpenAI Evals，这是他们的软件框架，用于创建和运行用于评估GPT-4等模型的基准。

在这里阅读所有关于它的信息。

但一个多星期后，更重要的消息传来了。

聊天-GPT 插件

任何软件，在获得合理的普及后，都会演变成一个平台。无论是Facebook，Microsoft Office还是Android，当工具开始提供插件或扩展时，它的真正力量就会释放出来！在工具之上构建的能力为开发人员社区打开了大门。ChatGPT 在 23 月 <> 日也做了同样的事情，当时他们宣布了一些他们一直在与合作伙伴合作的插件，还有一个 API 供您做同样的事情！*在非常抽象的层面上，这使 ChatGPT 能够通过调用第三方工具的 API 来执行基本操作。*它还可以通过它们读取数据来更新其知识库！

所以现在，你不会听到它抱怨它的知识有限到2021年，至少对于手头的特定任务而言。

第一个插件由Expedia，FiscalNote，Instacart，KAYAK，Klarna，Milo，OpenTable，Shopify，Slack，Speak，Wolfram和Zapier创建。他们还构建了一个浏览器和一个代码解释器插件。用他们自己的话说：

“虽然不是一个完美的类比，但插件可以成为语言模型的”眼睛和耳朵“，让它们访问太新、太个人化或太具体而无法包含在训练数据中的信息。

这将极大地扩展工具的功能：

  • 聊天 GPT 将超越输出文本！
  • 这可以通过赋予它访问最新信息的能力来解决“幻觉”问题。
  • 向开发者社区开放，以便通过聊天 GPT 进行进一步创新。
  • API 现在可以由 AI 构建和使用！
  • 它可以聚合来自不同插件的数据，为用户执行最佳操作。

如果您是开发人员，这将改变您对将LLM与工具集成的方式。它还改变了产品的构建和使用方式。让我们举一个例子来说明这一点。

想象一下，您正在计划去欧洲旅行。预订机票和酒店以及制定行程是您可以使用多个网站/工具完成的任务。您还需要知道景点是什么，旅行的最佳时间是什么，天气等等。一个摩擦点是用户需要浏览多个网站，了解他们的导航和用户体验，并在接听电话之前汇总所有这些。

我应该预订长周末的机票吗？等等，让我检查一下酒店的价格。哦，但是要下雨了！但是，有一个室内啤酒节。您需要汇总脑海中的所有场景。您是协调者。

现在，想象一下将所有这些交给助手，只有在这种情况下，助手才是一台机器，可以访问所有这些网站、数据，并且可以记住对话的上下文甚至一些历史记录！这就是插件可以为 ChatGPT 带来的强大功能。你可以想象如果有亚马逊或Facebook插件的可能性！ 但是*，*这带来了有关隐私和安全的各种担忧。因此，“负责任的AI”是OpenAI和其他公司发布这些公告的最前沿。
